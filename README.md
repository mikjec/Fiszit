# FiszIt

FiszIt is a web application for learning vocabulary with flashcards.  
The project was built with Laravel and provides a simple way to create, manage and review flashcards in a structured learning environment.

## Features

- user authentication and authorization
- creating, editing and deleting flashcards
- managing personal flashcard collections
- clear and simple user interface
- vocabulary learning support in a web-based format

## Tech Stack

- PHP
- Laravel
- Blade Templates
- Eloquent ORM
- MySQL
- HTML
- CSS
- JavaScript

## Purpose of the Project

The goal of this project was to practice fullstack web development with Laravel and build an application with real, practical functionality.  
It was also an opportunity to work with authentication, database relationships and CRUD operations in a complete web application.

## Project Structure

The application includes:

- authentication system
- flashcard management logic
- database models and relationships
- server-side rendered views using Blade templates
